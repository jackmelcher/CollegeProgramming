EECS40 - Assignment 2 - Dash-Till-Puff Game

Features:
Spaceship that accelerates up when a finger is held on the screen and 
accelerates down when finger is removed from screen

Trajectory line that guides the spaceship on it's journey

Cosmic Objects spawn above and below the trajectory line

If spaceship crashes into cosmic object, the game restarts

Added Features:
A score counter to give player an incentive to play

A start screen before the game plays
